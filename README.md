# 98 여행 일정 생성기
Windows 98 스타일의 ChatGPT 연동 여행 일정 생성 사이트


## 프로젝트 소개
[98.css](https://github.com/jdan/98.css)를 이용하여 Windows 98 스타일의 여행 일정 생성 사이트를 만들었습니다.
